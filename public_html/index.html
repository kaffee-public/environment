Environment project

e00

The zeroth environment is made up of projects only; as they are managed by the IDE.

The default IDE to use is NetBeans. I've chosen NetBeans at this point because of it's simplicity and out-of-the-box nature.

Each project has it's own build/dist/lib folders which is not optimal.

Installation

No specific install; use NetBeans to create projects and their folders.

Removal

Simply delete the folder e00 from your system.
